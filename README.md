# 计算机2105班上课源码

## git项目初始化的步骤

- 1:在gitee（或者其它的git服务商网站）上创建仓库
- 2:获取仓库的ssh下载地址
- 3:在想要下载的文件中打开git命令窗口，执行`git clone 仓库地址`下载项目

## git项目的版本提交和推送到服务器的步骤

- 1:对项目文件进行修改（添加/删除/修改）
- 2:vscode切换到源代码管理视图，对修改的文件进行添加操作
- 3:添加修改的内容说明
- 4:点击提交，完成版本记录
- 5:在终端中执行`git push`将版本记录推送到服务器
